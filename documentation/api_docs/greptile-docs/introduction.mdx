## Getting Started

<Steps>
    <Step title="Start your Greptile free trial">
        Sign Up [here](https://app.greptile.com/signup) using email, Google, Github, or GitLab.
    </Step>
    <Step title="Connect your GitHub or GitLab account">
        Using GitHub App or GitLab Access Tokens.
    </Step>
    <Step title="Enable Greptile on your repositories">
        Enable Greptile on the repositories you'd like to review.
    </Step>
    <Step title="That's it!" icon="party-horn">
        Greptile will now review all new pull/merge requests!
    </Step>
</Steps>



<Note>
        Greptile doesn't run on past PRs by default, but you can trigger the bot by commenting `@greptileai` on the PR.
</Note>
